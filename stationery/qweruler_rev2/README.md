# QWERuler 取扱説明書
QWERuler (基板定規) を購入していただきありがとうございます。

## 1. 注意事項
本製品は精密な計量を目的としたものではありません。精密な計測のためには必ずJIS規格などに適合した定規を使用ください。

以下は、実装キットを購入された方への説明書となります。

> [!WARNING]
> 本説明は**2025年度エンジニアフェスティバル in 東大以降で販売していた**QWERuler実装キットの説明です。
> 
> 2025年度五月祭・駒場祭で購入された方は[こちら](../qweruler/README.md)をご覧ください。

本製品はDIYキットです。特に、一部バージョンの製品を組み立てる際にははんだごてを含む怪我の危険性のある機器を使用する必要があります。十分注意して作業を行うようにしてください。

また、本製品は完成後、コンピュータ等に接続することで、USBキーボードとして使用することができますが、本製品の使用によって発生するいかなる損失についても、本製品の使用者は自己責任とします。

## 3. 組み立て方

### 1. ピンヘッダを取り付ける
ピンヘッダは黄色の棒が沢山刺さっているパーツです。今回は長い2本のみを使います。
これをもころん(犬のキャラクター)がいるほうに黄色と短い棒がくるようにして差し込みます。

![](../assets/qweruler_rev2/pinheader_1.jpg)

反対側をはんだ付けします。このとき、必要なのは5ピンだけで、固定用と合わせて6つはんだ付けすれば十分です。

![](../assets/qweruler_rev2/pinheader_22.png)

はんだ付けのやり方については、[イチケンさんの動画](https://www.youtube.com/watch?v=dQ7AUjb1tkA)がわかりやすく、おすすめです。

まず、1ピンだけはんだ付けして、位置合わせをします。

このようにずれてしまった時は、はんだごてを先ほどはんだ下部分に当てながら、指で他のピンを抑えて正しい位置にします。
![](../assets/qweruler_rev2/pinheader_3.jpg)
![](../assets/qweruler_rev2/pinheader_4.jpg)

はんだ付けが終わった後はこのようになります。
![](../assets/qweruler_rev2/pinheader_5.jpg)

### 2. マイコンを取り付ける
RP2040-zeroを取り付けます。もころんがいるほうに、USBが上に来るように、差し込みます。
![](../assets/qweruler_rev2/rp2040zero_1.jpg)

必要な部分をはんだ付けします。0,7,8,14,15,GNDとかかれた箇所をはんだ付けすれば十分です。

![](../assets/qweruler_rev2/rp2040zero_2.jpg)

### 3. ダイオードをはんだ付けする
ダイオード(小さい部品)を穴に差し込みます

> [!WARNING]
> 必ず向きに注意してください！

このとき、黒い線が入っているほうが、基板の白い線が入っているほうにします。
![](../assets/qweruler_rev2/diode_1.jpg)

すべて向きに注意して差し込んでください
![](../assets/qweruler_rev2/diode_2.jpg)

差し込んだら、反対側からはんだ付けします。
はんだ付け後、ダイオードのほうにまではんだが出てくるくらいはんだを使うのが良いはんだ付けです。
![](../assets/qweruler_rev2/diode_3.jpg)

最後に、余計な線をニッパーで切ってください。

### 4. スイッチをはんだ付けする
スイッチの金色の2つの金属をもころんが書いてある方から差し込みます
![](../assets/qweruler_rev2/switch_1.jpg)

裏側からはんだ付けします。このとき、付属しているスイッチが3ピン(金属のピンと真ん中の白いピンしかない)の場合、スイッチの位置が完全に固定されていないため、1ピンはんだ付けして、位置調整をしながらはんだ付けするようにしてください。

![](../assets/qweruler_rev2/switch_2.jpg)

こんな感じで片方ずつはんだ付けする。
![](../assets/qweruler_rev2/switch_3.jpg)

位置が良さそうであれば両方のピンをはんだ付けして終了です。

### 5. 組み立てをする
もう1本の基板定規に、スペーサーとネジを固定します。
向きはネジが目盛り側、スペーサーが部品接地面側です。

ネジをしたから差し込みます。
![](../assets/qweruler_rev2/screw_1.jpg)

差し込む場所はQFN-16の上下と、QFN-64の上下です。
![](../assets/qweruler_rev2/screw_2.jpg)
![](../assets/qweruler_rev2/screw_3.jpg)

スペーサーを反対側から手で回して取り付けます。
![](../assets/qweruler_rev2/screw_4.jpg)

上から本体を取り付けて、ドライバーでネジを回します。
![](../assets/qweruler_rev2/screw_5.jpg)

最後に、キーキャップを取り付けて完成です！
![](../assets/qweruler_rev2/complete.jpg)

### 5. ファームウェアの書き込み

まず、[ここ](https://github.com/uNikks/Pentronic-Lab/releases/tag/qweruler_rev2)から、QWERuler rev2のファームウェア(`qweruler_rev2_vial.uf2`)をダウンロードします。

パソコンとUSB-Cで接続します。

PC上にフォルダ（ドライブ）が開かない場合：接続した状態で、USB端子の右下にあるRESETボタンを2回押してください。「RPI-RP2」というドライブとして認識されます。

先ほどダウンロードしたファイルを、「RPI-RP2」ドライブ（エクスプローラー）へドラッグ＆ドロップしてください。 自動的に再起動し、キースイッチを押したときに、QWERと打たれたら成功です！

## 6. キーマップの変更
qwerでは役に立たないので、割り当てるキーを変更しましょう。

QWERuler rev2のファームウェアははじめからvialに対応しています。

[vial](https://vial.rocks/)にアクセスし、キーマップを変更してください。

vialの詳しい使用法については、[サリチル酸さんの記事](https://salicylic-acid3.hatenablog.com/entry/vial-manual)をご覧ください。
